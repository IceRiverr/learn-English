# Sample sources

## Context engineering with Dex Horthy

`context-engineering-with-dex-horthy.mp3` is the audio supplied locally for
The Pragmatic Engineer podcast episode “Context engineering with Dex Horthy”
(July 15, 2026).

- Episode: https://newsletter.pragmaticengineer.com/p/context-engineering-with-dex-horthy?showTranscript=true
- Publisher: The Pragmatic Engineer
- Host: Gergely Orosz
- Guest: Dex Horthy

`context-engineering-with-dex-horthy.json` was extracted from the episode's
public Substack transcript. Its word-level data was grouped into roughly
15-second passages to match the transcript layout shown on the episode page
and the minimal JSON format used by this app.
