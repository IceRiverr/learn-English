# Private course workspace

1. Fill in `course.local.json` before downloading or transforming sources.
2. Keep original assets under `source/`.
3. Generate the English draft as `course.en.json`.
4. Run `scripts/translation-workflow.mjs prepare` into `translation/`.
5. Record manual transcript, speaker, and translation QA under `review/`.
6. Do not copy anything to `public/` until the rights and quality gates in
   `../README.md` are satisfied.
